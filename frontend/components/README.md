# frontend/components

## Purpose
Reusable React components for Deno-based frontend (charts, controls, widgets).

## Suggested Initial Files
- **Button/**: For reusable button components.
- **Chart/**: For financial chart components using React-Financial-Charts.
- **TradeForm/**: For trade-related form components.
- **index.tsx**: Exporting shared components.

## Notes
- Keep components small, well-documented, and with unit/display tests.
- Use Material-UI for consistent styling and pre-built components.
- Ensure components integrate seamlessly with the main content area and sidebars.
